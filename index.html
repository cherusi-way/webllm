クラウドに送らない、
あなたの端末のAI。

モデルをこの端末にダウンロードして、ブラウザの中だけで動かします。通信が必要なのは初回だけ。一度読み込めば、オフラインでもキャラクターと会話できます。

準備中…0%

初回はモデル（数百MB〜数GB）のダウンロードに数分かかることがあります。2回目以降は端末のキャッシュから瞬時に起動します。WebGPU対応ブラウザ（PCの Chrome / Edge など）が必要です。

この端末では WebGPU が使えません。
Ember はブラウザ内でモデルを動かすために WebGPU を使います。最新の Chrome または Edge（PC推奨）でお試しください。
Firefox では about:config の dom.webgpu.enabled、Safari は最新版で有効になる場合があります。
結果をどのキャラクターに送りますか？ 選ぶとチャットの入力欄に貼り付けます。

キャラクターを選ぶ

話し相手を選んでください。いつでも切り替えられます。モデルはそのまま、性格だけが変わります。

道具箱

チャット以外の使い方。同じモデルのまま、文章を処理したり挙動を試したりできます。

会話の履歴

過去の会話を選ぶと続きから話せます。会話は送信するたびに自動保存されます。
